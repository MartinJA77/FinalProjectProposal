The Spellbook Project

Primary Project Features:

  • Displaying a main bank of spells.
  • Some form of user input to select particular spells add to spellbook.
  • User input to access spellbook.
  • Spellbook which stores selected spells and allows for spell information to be accessed.
  • Some form of user input which allows spell information to be accessed before adding to the spellbook.
  • Ability for a user to save a spellbook that can be loaded in a future instance of the application.

Secondary Project Features:

  • Ability to create and access multiple spellbooks.
  • Ability for a user to input a custom spell.
  • Ability for the information of spells to be adjusted within the spellbook.
  • Some sort of encryption for the imported text file

Project Overview


The goal of this project will to create a Python or web application (currently undecided) with which a user will be able to select different spells from a general list of all possible spells. Selecting these spells will save them to the user's spellbook, which they will be able to access in order to obtain various pieces of information about the spell. Additional possible features that are considered extra would be to create multiple spellbooks, be able to add custom spells, and perhaps possible ways to manipulate the spells. This project will utilize Python, and possibly HTML and CSS.


Technical Aspects of the Project


Although as it stands I am uncertain of the full extent of the technical aspects required for this project I have some ideas to begin with. For example, all of the spells will be imported from a text file that contains all of that information, which parses a particular line when viewing that spells information (so a particular line, or set of lines will be an address for that spells information). The spellbooks may be stored as a series of numbers that correspond to each spell, perhaps in an array.


Additional Technical Aspects


Some features that will be implemented, or goals to be implemented may utilize information that is not covered within the class. Some examples of such features are encryption, and application windows or python on a web page.
